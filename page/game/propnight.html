PropNight

Propnight é um jogo multiplayer baseado no jogo das "hide-and-seek" com terror clássico de sobrevivência, onde quatro jogadores enfrentam um único jogador que os tenta encontrar.

A diferenças entre estes dois jogos é: os jogadores que escondem conseguem transformar em objetos perto deles para se esconderem melhor, e quem procura tem também poderes especiais para os encontrar.

Requisitos minimos de Sistema

Sistema Operativo Windows 10 de 64-bit
Processador Intel Pentium G5905 3.3.50ghz 4mb Lga 1200
Memória 8 GB de RAM
Placa gráfica Geforce GTX 960 ou uma AMD equivalente
DirectX Versão 11
Armazenamento 15 GB de espaço disponível

PropNight

Propnight é um jogo divertido e único para aqueles que procuram uma experiência diferente de jogo multiplayer. Os Props escondem-se dos Caçadores, enquanto os Caçadores tentam encontrar e eliminar todos os Props antes do tempo acabar. No entanto, a jogabilidade pode ser um bocado suja o que não agradável para alguns jogadores.
